# 养蜂コミュニティ Backend

このディレクトリには、養蜂コミュニティサイトのバックエンド（CMS）が含まれています。

## 機能

- 养蜂家情報の管理
- はちみつ商品データの管理
- ユーザー認証と権限管理
- APIエンドポイント提供

## 技術スタック

- CMS: Strapi v4
- Database: SQLite（開発用）/ PostgreSQL（本番用）

## 開発

1. 必要なパッケージをインストール: `npm install`
2. 開発サーバーを起動: `npm run develop`
3. 管理画面: `http://localhost:1337/admin`
